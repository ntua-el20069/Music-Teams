Welcome to Music Teams (Demo) API

The functional API endpoints are listed below


Home page with all songs of the demo team (GET, POST)

Add song (GET, POST)

Add chords (GET, POST)
    Example: /API/4/add-chords
   parameter: song_id (in the example it is 4)

Song Transpose (GET, POST)
    Example: /API/4/song-transpose
   parameter: song_id (in the example it is 4)

Web Scrape from Given HTML (Greek Lyrics) (POST)
    /API/webscrape
    POST body: HTML from greek lyrics site (text/html)

Upload Recording file (POST)
    Example: /API/4/upload
    request.files   Recording File

Make Song Demand (POST)
    /API/make-song-demand
    POST body:   JSON with a field 'title'

Recent Song Demands (Most recent first) (GET)
